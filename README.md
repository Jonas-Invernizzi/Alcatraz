# Alcatraz
Desenvolvido durante o segundo ano do Ensino Médio Técnico em Informática, o Alcatraz desafia o jogador a escapar da prisão e alcançar o barco de fuga sem ser capturado pelos guardas.

O jogo foi criado com HTML, CSS, JavaScript e PHP, combinando design limpo, lógica interativa e persistência de dados. O HTML organiza os elementos do jogo, o CSS cuida do visual, o JavaScript gerencia os movimentos e interações, e o PHP controla pontuações e progresso.

Este projeto demonstra habilidades em programação web, trabalho em equipe e planejamento, aplicando conhecimentos técnicos em um jogo estratégico, divertido e desafiador.
