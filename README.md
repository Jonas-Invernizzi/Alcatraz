# Alcatraz
Criado durante o segundo ano do Ensino Médio Técnico em Informática, este jogo desafia o jogador a escapar da prisão e alcançar o barco de fuga sem ser capturado pelos guardas. Desenvolvido com HTML, CSS, JavaScript e PHP, combina estratégia, ação e raciocínio rápido em uma experiência divertida e desafiadora.
